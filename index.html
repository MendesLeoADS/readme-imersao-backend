IMERSÃO BACKEND

O repositório que você irá acessar agora, é referente uma aplicação backend fruto de uma participação do evento online chamado "Imersão Backend" da empresa de tecnologia ALURA.

Nesta aplicação desenvolvemos e criamos uma API RestFul do zero, com integração a um banco de dados na nuvem, o MongoDB, utilizamos bibliotecas e frameworks do JavaScript, como NodeJs, ExpressJs. Controlando suas versões com GIT e utilizando ferramentas como Thunder Client e Postman, para realizarmos alguns GET, POST e PUT na API.

Outra coisa legal de mencionar, é que adicionamos essa nossa aplicação a Google Cloud. Isto mesmo, a nuvem do Google. Foi uma experiência enriquecedora que desejo compartilhar com vocês.
